Copies the active tab's URL to clipboard with common tracking and non-essential parameters stripped. The icon matches Firefox's built-in toolbar style, turns green for 3 seconds to confirm the copy.

Removes UTM parameters (utm_source, utm_medium, utm_campaign, etc.), ad click IDs (fbclid, gclid, msclkid), analytics tags, YouTube extras (list, t, index, feature), and other common tracking fields.

Binds Ctrl+Shift+C by default (rebindable via Manage Extension Shortcuts).
