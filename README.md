Copies the active tab's URL to clipboard with common tracking and non-essential parameters stripped. The icon matches Firefox's built-in toolbar style.

Removes UTM parameters (utm_source, utm_medium, utm_campaign, etc.), ad click IDs (fbclid, gclid, msclkid), analytics tags, YouTube extras (list, t, index, feature), and other common tracking fields.

Usage: click the toolbar button or press Ctrl+Shift+C (rebindable via Manage Extension Shortcuts). The icon briefly turns green to confirm the copy.
